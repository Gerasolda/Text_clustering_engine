
# Sliding Window Text Clustering

## Описание

Проект представляет собой алгоритм обработки текста, который выполняет кластеризацию слов на основе правил схожести в пределах скользящего окна фиксированного размера `k`.

Основная идея заключается в том, чтобы анализировать локальный контекст текста и объединять слова, которые считаются похожими по заданному эвристическому правилу. Для хранения и объединения групп используется структура данных **DSU (Disjoint Set Union)**.

---

## Как работает алгоритм

1. Входной текст разбивается на слова
2. Каждое слово нормализуется (приведение к нижнему регистру, удаление лишних символов)
3. Каждому уникальному слову присваивается ID
4. Поддерживается скользящее окно размера `k`
5. Все слова внутри окна сравниваются попарно
6. Если слова считаются похожими — объединяются через DSU
7. После обработки всего текста считаются частоты слов
8. Формируются группы и выводится результат

---

## Правило схожести слов

Используется эвристическая функция:

* одинаковая длина → допускается 1 различие символа
* разная длина → допускается разница в 1 символ при простом добавлении/удалении (`s` или `e`)

---

## Возможности

* Обработка текстовых данных
* Анализ локального контекста (sliding window)
* Кластеризация слов по схожести
* Объединение групп через DSU
* Подсчёт частот слов
* Сортированный вывод результатов

---

## Стек технологий

* C++
* STL

---

## Сложность

* Время: **O(n · k²)**
* Память: **O(n)**

---

## Ограничения

* Эвристическая функция схожести может давать неточные результаты
* Не оптимизировано для больших текстов
* Квадратичные сравнения внутри окна могут быть медленными

---

## Примечание

Проект является учебным алгоритмическим решением для изучения структур данных (DSU) и обработки текста с использованием скользящего окна.
